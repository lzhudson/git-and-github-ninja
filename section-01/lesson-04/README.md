# Trablahando com mais arquivos

## Comandos

```git add file-1, file-2, file, ...```

É possivel adicionar diversos arquivos de uma vez colocando o nome dos arquivos de forma sequencial.

```git add .```

No comando acima adiciona todos os arquivos que foram modificados na Staging Area.

```git diff --staged```
Compara as alterações da Staging Area com as alterações do .git directory.

Obs: o ```git diff``` por padrão compara a Working Directory com a Staging Area.
